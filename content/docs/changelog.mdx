# Welcome to OpenAI Realtime Blocks
## The UI library for the OpenAI Realtime API. 
Drop animated components that react to voice into your Nextjs app. [Get started](/docs)

## Voice AI Components.
All components are designed to react based on volume levels from the provided "use-webrtc" hook.
